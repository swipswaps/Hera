Meteorology
=======================

The meteorological module is used to manage meteorological data retrieved from different sources.
The main use is to manage all meteorological data in terms of loading and reading it from and to the database,
loading post-processed data to shorten computation and read times and to
specify all additional uses for meteorological data.
